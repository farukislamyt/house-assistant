# House Assistant - Manage Smartly

A professional, full-featured property management system that runs entirely in the browser with zero backend costs. Perfect for landlords and property managers.

## 🚀 Features

- **Complete Authentication System**: Secure signup/login with LocalStorage
- **Tenant Management**: Full CRUD operations, status tracking, lease management
- **Financial Tracking**: Rent payments, expenses, revenue analytics
- **Maintenance Requests**: Work order management with priority levels
- **Utility Tracking**: Monitor electricity, water, gas, internet usage
- **Document Storage**: Upload and manage important documents
- **Calendar View**: Payment schedules and lease dates
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Data Persistence**: All data stored locally in browser
- **Professional UI**: Modern glassmorphism design with smooth animations

## 🛠️ Tech Stack

- **Frontend**: Pure HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS (via CDN)
- **Charts**: Chart.js (via CDN)
- **Icons**: Font Awesome 6
- **Storage**: LocalStorage with JSON serialization
- **Security**: Client-side encryption for sensitive data
